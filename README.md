# 📊 Project Overview

This project analyzes a dataset of 200 students to understand the probability of passing a competitive mathematics examination.

The analysis covers:

- Probability Fundamentals
- Empirical vs Theoretical Probability
- Random Variables
- Probability Distributions
- Venn Diagrams
- Contingency Tables
- Joint Probability
- Marginal Probability
- Conditional Probability
- Bayes Theorem
- Relationship Analysis

The entire project was implemented using Python in Jupyter Notebook.

---

# 🗂 Dataset Features

| Feature | Description |
|----------|-------------|
| study_hours | Weekly study hours |
| attendance | Attendance percentage |
| group_discussion | Participation in group discussions |
| previous_test_score | Internal test marks |
| final_exam_pass | Pass or Fail |

# 📈 Analysis Performed

## 1. Understanding Probability

- Probability Concepts
- Sample Space
- Events
- Probability Examples

## 2. Types of Events

- Empirical Probability
- Theoretical Probability

## 3. Random Variable & Probability Distribution

- Binomial Distribution
- Mean
- Variance

## 4. Venn Diagram Analysis

- Students studying >10 hours
- Students with attendance >80%
- Overlapping students

## 5. Contingency Table Analysis

- Group Discussion vs Exam Result

Calculated:

- Joint Probability
- Marginal Probability
- Conditional Probability

## 6. Relationship Analysis

Determined whether:

- Independent Events
- Dependent Events
- Mutually Exclusive Events

## 7. Bayes Theorem Application

# 📊 Key Findings

✅ Students with high attendance have a greater probability of passing.

✅ Group discussion participation increases pass probability.

✅ Higher study hours positively impact performance.

✅ Attendance and group discussion participation are important predictors of success.
